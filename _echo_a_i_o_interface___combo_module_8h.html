enum AIOComboModuleParameters
Enumerator
AIO_comboParameterFirmwareVersion 

Read-only integer value of the firmware version, or -1 on error.

AIO_comboParameterSerialNumber 

Read-only integer value of the serial number, or -1 on error.

AIO_comboParameterAuxOut 

Integer parameter for the AIO-C AUX OUT pins; bits 0-7 of the value are a bit mask corresponding to the state of the AUX OUT pins.

AIO_comboParameterAuxIn 

Read-only integer parameter for the AIO-C AUX IN pins; bits 0-7 of the value are a bit mask corresponding to the state of the AUX IN pins.

AIO_comboParameter5VDCEnable 

Integer parameter to enable or disable the 5 VDC power supply; 0 to disable, 1 to enable.

AIO_comboParameterVariableDCPowerEnable 

Integer parameter to enable or disable the variable DC power supply; 0 to disable, 1 to enable.

AIO_comboParameterVariableDCPowerTargetMillivolts 

Integer parameter to set the target voltage in millivolts for the variable DC power supply, from 600 mV to 5000 mV.

AIO_comboParameterVariableDCPowerMeasuredMillivolts 

Read-only integer parameter for the actual measured voltage for the variable DC power supply.

AIO_comboParameterVariableDCPowerMeasuredAmperes 

Read-only double precision floating point parameter for the measured output current in amperes for the variable DC power supply.

AIO_comboParameterMeasuredCurrentRange 

Integer parameter to set the variable power supply current measurement range; must be one of the AIO_COMBO_MODULE_CURRENT_MEASUREMENT constants (see below)

AIO_comboParameterOverCurrentThresholdAmperes 

Double precision floating point parameter for the over current threshold in amperes; note that the value range for this parameter is determined by the measured current range.

AIO_comboParameterOverCurrentCondition 

Integer parameter for over current condition; if this parameter value reads as true, then then AIO-C module has detected an over current condition. Set this parameter to zero to clear the over current condition.
