15/*
16 AIO_tdmParameterFirmwareVersion
17 Read-only integer parameter for the AIO-T module firmware version; reads -1 on error.
18
19 AIO_tdmParameterClockMode
20 Sets the AIO-T module to clock source or clock sink mode; 0 - clock source, 1 - clock sink
21
22 AIO_tdmParameterBitsPerFrame
23 Sets the number of bits per TDM Frame; 01 - 64 bits/frame(not implemented), 10 - 128 bits/frame(not implemented), 11 - 256 bits/frame.
24
25 AIO_tdmParameterBitsPerWord
26 Sets the number of bits per TDM Word; 01 - 24 bits/word, 10 - 32 bits/word.
27
28 AIO_tdmParameterInvertSCLK
29 Integer parameter to invert the SCLK signal; 0 - to Data and FSYNC clocks out on the falling edge of SCLK, 1 - Data and FSYNC clocks out on the rising edge of SCLK.
30
31 AIO_tdmParameterAudioDataShiftEnabled
32 [Clock source mode] Integer parameter that delays the sampling of input SHIFT bits; 0 - TDM input sampling is aligned with TDM input, 1 - TDM output is advanced "SHIFT" bits adead of the TDM input.
33 [Clock sink mode] Integer parameter that enables SCLK output on BNC connector(Only valid for versions prior to 2.01. Must be set for versions 2.01 thru 2.0e, ignored for 2.0f and above)
34 0 - TDM output is aligned with TDM input, 1 - TDM output is advanced "SHIFT" bits ahead of the TDM input.
35
36 AIO_tdmParameterAudioDataShiftBits,
37 Integer parameter that set the number of bits to delay INPUT (clock source mode) or advance OUTPUT (clock sink mode) when SHIFT_EN is set. (7 max, 0 is interpreted as 1 for backward compatibility)
38
39 AIO_tdmParameterFSYNCPhaseDelay,
40 [Clock source mode] Integer parameter that determines whether FSYNC clocks out along with data or is delayed by 1/2 SCLK cycle; 0 - FSYNC clocks out along with data, 1 - FSYNC is delayed by 1/2 SCLK cycle
41 [Clock sink mode] Integer parameter that determines whether INPUT is delayed by 1/2 SLCK cycle; 0 - INPUT is sampled normally, 1 - INPUT sampling is delayed by 1/2 SCLK cycle
42
43 AIO_tdmParameterFSYNCPosition,
44 8 Bit parameter that sets the position for start of positive portion of FSYNC. Valid positions are 0
45
46 AIO_tdmParameterFSYNCWidth
47 8 bit integer parameter that sets the bit position for the start of positive portion of FSYNC. Valid positions are 0 (first data bit position) through frame length - 1
48 [clock source mode] bit 0 corresponds to the MSB of TDM output. [clock sink mode] bit 0 corresponds to the MSB of TDM input
49*/
